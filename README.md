# Kitabu - Digital Library Management System

![Kitabu Logo](public/egbook.jpeg)

Kitabu is a modern, premium digital bookstore and library management system designed for seamless book browsing, purchasing, and management. Built with a focus on visual excellence and performance.

## ✨ Key Features

- **📚 Comprehensive Catalog**: Search and filter books by category, new releases, and more.
- **🔐 Secure Authentication**: Full login and registration system for users.
- **🛒 Dynamic Shopping Experience**: Integrated shopping cart and streamlined checkout flow.
- **📂 User Library**: Personal "My Library" space to access and download digital books.
- **🛠️ Admin Dashboard**: Robust management interface for handling books, user accounts, and customer orders.
- **🌗 Dark Mode Support**: Sleek, theme-aware UI using modern design principles.
- **📱 Responsive Design**: Fully optimized for mobile, tablet, and desktop viewers.

## 🚀 Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router & Turbopack)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/) & [Radix UI](https://www.radix-ui.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Forms**: [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)
- **Animations**: [Embla Carousel](https://www.embla-carousel.com/) & [Tailwind Animate](https://github.com/jamiebuilds/tailwindcss-animate)

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.17 or later)
- [pnpm](https://pnpm.io/) (recommended) or npm/yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Elvinoacer/lib_management_system_mini.git
   cd lib_management_system_mini
   ```

2. **Install dependencies**:
   ```bash
   pnpm install
   ```

3. **Run the development server**:
   ```bash
   pnpm dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the results.

## 📂 Project Structure

```bash
├── app/               # Next.js App Router (Pages, Layouts, API)
├── components/        # Reusable UI components & features
│   ├── ui/           # Radix-based shadcn components
│   ├── books/        # Book-specific components
│   └── admin/        # Admin dashboard components
├── hooks/             # Custom React hooks
├── lib/               # Utility functions & shared logic
├── public/            # Static assets (images, icons, fonts)
├── styles/            # Global styling and Tailwind config
└── next.config.mjs    # Next.js configuration
```
